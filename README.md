# 灶台 · 跟着步骤学做菜

> 分步图解 / 精确计时 / 零基础也能搞定的做菜教程网站

## 功能

- 8 道精选菜谱（家常菜、快手菜、汤羹、甜品、早餐）
- 分类筛选 + 实时搜索
- 分步骤展示，每步含描述、小贴士、计时器
- 步骤完成标记 + 进度条跟踪
- 键盘快捷键（左右箭头导航、ESC 返回）

## 技术栈

纯前端：HTML + CSS + 原生 JavaScript，无构建工具，零依赖。

## 本地预览

```bash
python -m http.server 8090
# 然后打开 http://localhost:8090
```

## 在线访问

通过 GitHub Pages 部署：https://你的用户名.github.io/仓库名/
