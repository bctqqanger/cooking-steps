## 做菜步骤网站 - 前端搭建
- 用户要求完成一个关于做菜步骤的网站，先做前端
- 技术方案：纯 HTML + CSS + 原生 JS，无需构建工具，`python -m http.server 8090` 本地预览
- 项目路径：D:/project/cookinhg
- 文件结构：
  - index.html — 单页应用，首页+详情页视图切换
  - css/style.css — 暖色调美食主题（橙红/暖棕/奶油色），响应式布局
  - js/data.js — 8道菜谱数据 + SVG插图生成
  - js/app.js — 菜谱浏览、分类筛选、搜索、步骤导航、计时器、进度跟踪
- 功能特性：
  - 8道菜谱（番茄炒蛋、红烧排骨、蒜蓉西兰花、紫菜蛋花汤、葱花鸡蛋饼、凉拌黄瓜、银耳红枣羹、蛋炒饭）
  - 6个分类筛选（全部/家常菜/快手菜/汤羹/甜品/早餐）
  - 菜谱搜索（按菜名或食材）
  - 分步骤展示，每步含描述、小贴士、计时器
  - 步骤完成标记 + 进度条
  - 计时器浮窗（倒计时 + 声音提醒）
  - 键盘快捷键（左右箭头导航、ESC返回）
  - 每道菜配原创 SVG 插图，无外部图片依赖
- 本地预览地址：http://localhost:8090/index.html
